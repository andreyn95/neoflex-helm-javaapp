Проектное задание по разделу Kubernetes
Задание 1. Деплоим приложение
1. Установите кластер Kubernetes на виртуальной машине Main. Лучше
сделать это через Kubespray.
2. На машине Infra лежит Docker образ приложения, которое вы починили в
разделе про Docker. Его нужно будет развернуть в вашем кластере. Для этого
создайте личный репозиторий на https://hub.docker.com/ и загрузите в него образ.
3. Напишите deployment.yaml и service.yaml, разверните с их помощью это
приложение в кластере Kubernetes, и опубликуйте его (сделайте доступным извне
кластера) через сервис типа NodePort.
4. Пробросьте порт вашего сервиса наружу при помощи kubectl port-forward
и зайдите в веб-интерфейс приложения по адресу
http://<адрес_кластера>:8080/demo/
Сделайте скриншот веб-интерфейса вашего приложения. На скриншоте должно
быть видно IP адрес и порт.
5. Увеличьте несколько раз количество реплик приложения, используя
последовательно механизмы:
 kubectl apply
 kubectl scale
 kubectl edit
После каждого изменения количества реплик сделайте скриншот вывода
применённой команды и kubectl get rs && kubectl get po.
В ответ нужно прислать.
1. Скриншот вывода команды `kubectl get nodes`
2. Скриншот манифестов deployment.yaml и service.yaml.
3. Скриншот выставленного через сервис типа NodePort приложения из
Вашего репозитория.
4. Скриншот веб-интерфейса вашего приложения. На скриншоте должно быть
видно IP адрес и порт.
5. Скриншоты, показывающие добавление реплик приложения.
Задание 2. Пишем Helm чарт
1. Продолжаем развивать наше приложение. Теперь нужно написать Helm-
чарт для него. Создайте папку, в которой создайте Chart.yaml и values.yaml.
Создайте подпапку templates. В ней создайте шаблоны deployment.yaml и
service.yaml. Используйте знание шаблонов Helm, чтобы вынести в values.yaml
все подходящие значения из манифестов.
2. Разверните в кластере этот Helm-чарт и выставьте его через сервис типа
NodePort. Сделайте скриншоты вывода команд helm:
• Список приложений Helm
• Описание вашего чарта через команды Helm
В ответ нужно прислать.
1. Хелм-чарт. Можно прислать в виде ссылки на ваш репозиторий, или в виде
архива, или в виде текстового документа с манифестами.
2. Скриншоты вывода команд Helm:
• Список приложений Helm.
• Описание вашего чарта через команды Helm.
